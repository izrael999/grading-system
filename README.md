# Grading System

My PHP/MySQL project for managing student grades.

## Setup

1. Copy folder to `C:\xampp\htdocs\`
2. Start Apache and MySQL in XAMPP
3. Open phpMyAdmin, create database `student_management`
4. Import `DATABASE.sql`
5. Open browser: `http://localhost/grading-system/`

## Login

- Admin: admin / admin123
- Teacher: teacher / teacher123  
- Student: student / student123

## Files

- `config.php` - database connection
- `login.php` / `register.php` - login/signup
- `dashboard.php` - main page
- `add_student.php` - add students
- `add_marks.php` - add marks
- `DATABASE.sql` - import this file

That's it. Everything should work.
